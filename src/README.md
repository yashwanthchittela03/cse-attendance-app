# CSE College Attendance Tracker

A modern, full-stack web application built for CSE students to track daily class attendance, calculate percentage statistics, and maintain real-time cloud data synchronization across all devices.

---

## 🚀 Features

- **Username Authentication**: Fast and simple login/registration powered by Supabase Auth (no email verification required).
- **Cloud Database Sync**: Real-time persistent storage via PostgreSQL on Supabase.
- **Custom Timetable Management**: Automated schedule mapping including 1-hour split periods, lab blocks, and evening OE/Minor course slots.
- **Manual Attendance Save**: Toggle daily class attendance with explicit **Save** and **Update** confirmation buttons.
- **Bulk Quick Actions**: Mark entire daily schedules as **Present** or **Absent** in a single tap.
- **Dark Mode UI**: Clean, mobile-friendly dark theme optimized for daily usage.

---

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Backend & Database**: Supabase (PostgreSQL & Authentication)
- **Deployment**: Vercel
